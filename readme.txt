== Description ==

A plugin to center and size Amazon SiteStripe images as well as add a CTA button. Width of image and color of button can be updated on the plugin settings page.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Plugin will work in the background on all pages with a SiteStripe image.
